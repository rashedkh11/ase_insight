# ASE Insight

## Project Overview
**ASE Insight** is a data analysis and machine learning project focused on providing insights and predictions for the **Amman Stock Exchange (ASE)**. The project includes an interactive **Streamlit dashboard** and Python tools that allow users to explore stock data, analyze trends, and make predictions based on historical and technical indicators. This project demonstrates skills in **data analysis, machine learning, and web app deployment**, making it an ideal portfolio project for a Junior ML Developer.

## Features
- Interactive dashboards for **AI-driven stock predictions** using Streamlit.
- Data visualization of ASE stock performance and technical indicators.
- Machine learning models for prediction tasks and analysis.
- Multiple pages for different functionalities:
  - **AI Prediction**: Predict stock trends based on input data.
  - **Technical Dashboard**: Visualize ASE stock data and technical indicators.
  - **Stock Chat**: Ask questions about ASE stocks and get automated insights.
  - **About & Help**: Learn about the project and how to use it.

## Tech Stack
- **Programming Language:** Python  
- **Web Framework:** Streamlit, Flask  
- **Data Analysis:** Pandas, NumPy  
- **Machine Learning:** Scikit-learn, TensorFlow / PyTorch (if used)  
- **Other Tools:** Excel for datasets

## How to Use
1. Open the app in your browser.
2. Navigate between pages using the sidebar:
   - **AI Prediction**: Input stock data to get trend predictions for ASE stocks.
   - **Technical Dashboard**: Explore charts and technical indicators for ASE stocks.
   - **Stock Chat**: Ask questions about ASE stocks and receive automated insights.
   - **About & Help**: Learn more about the project and how to use it.
3. Explore datasets, predictions, and visualizations interactively.
